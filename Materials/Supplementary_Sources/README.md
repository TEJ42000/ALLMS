## â• Adding New Materials

### Case Law
1. **Identify court**: ECHR, CJEU, or National Courts
2. **Place in appropriate directory**: `Case_Law/[Court]/`
3. **Use standard naming**:
   - Format: `CASE_OF_[Party]_v_[Party].pdf`
   - Example: `CASE_OF_SALDUZ_v_TURKEY.pdf`
4. **Include metadata**: Court, date, case number in filename or separate index

### External Resources
1. **Create or update** `External_Links.md`
2. **Organize by category**: Databases, Government Sites, Academic Resources
3. **Include descriptions**: Brief explanation of each resource
4. **Verify links**: Ensure URLs are current and accessible

### Additional Reading
1. **Place in** `Additional_Reading/` subdirectory
2. **Organize by topic** or subject area
3. **Include source information** in filename or metadata
4. **Verify copyright**: Ensure materials can be legally shared

## ğŸ“ File Naming Convention

### Case Law
**Good Examples:**
- `CASE_OF_SALDUZ_v_TURKEY.pdf`

- Generic names: `case1.pdf`
